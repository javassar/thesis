# How to Play "The Last Light"

## Quick Start

The game is **currently running** at: **http://localhost:5173/**

Just open your browser and go to that URL to play!

## If the server is not running:

```bash
cd the-last-light
npm run dev
```

Then open: **http://localhost:5173/**

## Controls

- **↑ / W** - Move selection up
- **↓ / S** - Move selection down
- **Enter** - Confirm / Continue
- **Space** - Advance text / Light the lamp

## About the Game

You are Marlowe, a lighthouse keeper of 34 years. Tomorrow, the lighthouse becomes automated. Tonight, you climb the tower one last time, making choices that shape your emotional journey and determine what legacy you leave behind.

**Playtime**: 2-5 minutes
