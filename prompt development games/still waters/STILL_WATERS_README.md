# STILL WATERS

A game about letting go.

## How to Run

Simply open `still-waters.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

```bash
# On macOS
open still-waters.html

# On Linux
xdg-open still-waters.html

# On Windows
start still-waters.html
```

Or double-click the file to open it in your default browser.

## Controls

- **Arrow Keys** or **WASD**: Move your cursor
- **R**: Restart the game
- **ESC** or **Q**: Quit

## How to Play

The game provides no instructions intentionally. Explore and discover what happens when you interact with the letters.

## Design Philosophy

This game explores grief, loss, and letting go through mechanics rather than words. Your instinct to help may not always be what's needed. Sometimes, the most loving thing you can do is to simply be present.

## Technical Details

- Platform: Web browser (HTML5 Canvas)
- Resolution: 800x600 pixels
- No external dependencies required
- Works offline once loaded
