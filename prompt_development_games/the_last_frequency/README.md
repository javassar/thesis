# The Last Frequency

A narrative thriller game created with Pygame.

## How to Play

### Prerequisites

- Python 3
- Pygame

### Installation

1.  **Install Pygame:**
    ```bash
    pip install pygame
    ```

2.  **Run the game:**
    ```bash
    python the_last_frequency/main.py
    ```

### Controls

-   **Left/Right Arrow Keys:** Adjust the frequency.
-   **Spacebar:** Interact (not used in the final version, but the hook is there).
