# The Last Signal

A 5-minute narrative puzzle experience about a radio operator receiving transmissions from the future.

## How to Play

### Running the Game

1. Open `index.html` in a modern web browser (Chrome, Firefox, Safari, or Edge)
2. For the best experience, use fullscreen mode (F11 on most browsers)
3. Make sure your browser allows audio playback

### Controls

- **Arrow Keys** or **A/D** - Tune the radio frequency dial
- **Letter Keys** - Type decoded Morse code messages
- **Number Keys 1-3** - Select dialogue choices
- **H** - Show Morse code hint during decoding
- **Space** - Start the game from intro screen
- **R** - Replay after reaching an ending

### Objective

You are Sam Reeves, a radio operator at Station Vostok-7 in the Arctic on December 14, 1962. A mysterious signal breaks through the static. Your goal is to:

1. Tune to frequency 107.7 MHz to lock onto the signal
2. Decode the Morse code transmission
3. Make critical choices based on the information you receive
4. Decide whether to trust the voice on the other end

### Features

- **7+ Dynamic Ending Variations** - same choices lead to different outcomes based on hidden psychological state
- **Hidden Trust & Reality Systems** - your belief/doubt affects the narrative in subtle ways
- **Temporal Paradox Mechanics** - experience time echoes and reality fragmentation
- **Secret Fourth Ending** - unlock by achieving maximum reality fragmentation
- **Branching narrative** - your decisions matter in visible AND invisible ways
- **Morse code puzzle** - with temporal glitches if reality becomes unstable
- **Secret frequency** - try tuning to 107.3 MHz early in the game
- **Meta-narrative layer** - the game occasionally questions its own nature
- **Atmospheric 1960s aesthetic** with authentic analog interface design

### Tips

- Pay attention to the signal strength meter (VU meter) when tuning
- The frequency dial requires precision - fine-tune to lock the signal
- During Morse code, press H if you need a reference guide
- There are no fail states - different choices lead to different story paths
- The game takes approximately 4-6 minutes to complete
- Try replaying with different belief systems (trust everything vs. doubt everything)
- Watch for temporal echoes and reality glitches
- Extreme skepticism unlocks a secret fourth ending!

### Technical Notes

- No installation required - runs entirely in the browser
- Uses Web Audio API for sound synthesis
- Optimized for 1280x720 resolution and above
- Keyboard-only controls (no mouse required for gameplay)
