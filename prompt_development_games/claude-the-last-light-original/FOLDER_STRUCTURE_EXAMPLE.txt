═══════════════════════════════════════════════════════════════
  EXPERIMENT FOLDER STRUCTURE - Copy This Exactly
═══════════════════════════════════════════════════════════════

my-experiment/                          ← Your main experiment folder
│
├── index.html                          ← Renamed from experiment-template.html
│                                         This is your main experiment file
│                                         Participants will open this URL
│
└── games/                              ← Folder containing all game builds
    │
    ├── the-last-light-revised/         ← Game 1 folder
    │   ├── index.html                    ← Copy from dist/index.html
    │   └── assets/                       ← Copy entire assets folder from dist/
    │       ├── index-BaDgWKMd.css
    │       └── index-kpetSLys.js
    │
    ├── the-last-light-original/        ← Game 2 folder (if you have it)
    │   ├── index.html
    │   └── assets/
    │       ├── index-xxxxx.css
    │       └── index-xxxxx.js
    │
    └── another-game/                   ← Game 3 folder (if you have it)
        ├── index.html
        └── assets/
            ├── index-xxxxx.css
            └── index-xxxxx.js


═══════════════════════════════════════════════════════════════
  HOW TO CREATE THIS STRUCTURE
═══════════════════════════════════════════════════════════════

STEP 1: Create the folders
────────────────────────────────────────────────────────────────
mkdir my-experiment
mkdir my-experiment/games
mkdir my-experiment/games/the-last-light-revised


STEP 2: Copy the experiment file
────────────────────────────────────────────────────────────────
cp experiment-template.html my-experiment/index.html


STEP 3: Build and copy your game
────────────────────────────────────────────────────────────────
cd the-last-light-revised
npm run build

# This creates: the-last-light-revised/dist/

# Now copy EVERYTHING from dist/ to games folder:
cp -r dist/* ../my-experiment/games/the-last-light-revised/


STEP 4: Verify the structure
────────────────────────────────────────────────────────────────
cd ../my-experiment
ls -R

# You should see:
# index.html
# games/the-last-light-revised/index.html
# games/the-last-light-revised/assets/...


═══════════════════════════════════════════════════════════════
  WHAT EACH FILE DOES
═══════════════════════════════════════════════════════════════

my-experiment/index.html
│
├─► Main experiment file
├─► Contains jsPsych code
├─► Shows consent, instructions, survey
├─► Loads games in iframe
└─► This is what participants open


games/the-last-light-revised/index.html
│
├─► Your built React game
├─► Loaded inside iframe
├─► Signals when complete
└─► Sends data back to parent

═══════════════════════════════════════════════════════════════
  FOR MULTIPLE GAMES
═══════════════════════════════════════════════════════════════

If you have 3 games, structure looks like:

my-experiment/
├── index.html
└── games/
    ├── game1/
    │   ├── index.html
    │   └── assets/
    ├── game2/
    │   ├── index.html
    │   └── assets/
    └── game3/
        ├── index.html
        └── assets/

Then in index.html, configure:

const GAMES = {
    'game1': {
        name: 'The Last Light (Revised)',
        path: 'games/game1/index.html',
        description: 'Mystery version'
    },
    'game2': {
        name: 'The Last Light (Original)',
        path: 'games/game2/index.html',
        description: 'Original version'
    },
    'game3': {
        name: 'Another Game',
        path: 'games/game3/index.html',
        description: 'Different game'
    }
};

═══════════════════════════════════════════════════════════════
  IMPORTANT NOTES
═══════════════════════════════════════════════════════════════

❗ Always copy the ENTIRE dist/ folder contents, not just index.html
❗ The assets/ folder is required - it contains the CSS and JavaScript
❗ Paths in index.html must match your folder structure exactly
❗ Use relative paths (games/...) not absolute (/games/... or https://...)
❗ Test with python -m http.server, NOT by opening file:// directly
❗ All games must be in the same domain/folder for iframe communication
