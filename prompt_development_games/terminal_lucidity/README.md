# Terminal Lucidity

This is a game created based on a design document.

## How to Play

- **Move the mouse** to control the cursor.
- **Push words** with your cursor to move them.
- **Click and hold** on words to "delete" them by shrinking them.
- The goal is to arrange the words on the black shelf to form a message before the timer runs out.

## How to Run the Game

1.  You need a local web server to run this game because of browser security policies (CORS).
2.  If you have Python installed, you can use its built-in web server.
3.  Open a terminal or command prompt.
4.  Navigate to the `terminal_lucidity` directory.
5.  Run the following command:
    ```bash
    python -m http.server
    ```
6.  Open your web browser and go to the following address:
    [http://localhost:8000](http.localhost:8000)

The game will start automatically.
